AnimTrap Scroll Animation API

AnimTrap Scroll Features is exposed as a global variable, for now there are three methods available:
  • init - initialize AnimTrap Scroll
  • refresh - recalculate all offsets and positions of elements (called on window resize)
  • refreshHard - ireinit array with AnimTrap elements and trigger refresh (called on DOM changes that are related to aos elements)

    • Example execution:

    ANIM.refresh();

    By default ANIM is watching for DOM changes and if there are any new elements loaded asynchronously or when something is removed from DOM it calls refreshHard automatically. In browsers that don't support MutationObserve r like IE you might need to call ANIM.refreshHard() by yourself.

    refresh method is called on window resize and so on, as it doesn't require to build new store with ANIMTRAP elements and should be as light as possible.

    Global settings

    If you don't want to change setting for each element separately, you can change it globally.

    To do this, pass options object to init() function, like so:

        
         <script>
             Anim.init({
                 offset: 200,
                 duration: 600,
                 easing: 'ease-in-sine',
                 delay: 100,
              });
         </script>
